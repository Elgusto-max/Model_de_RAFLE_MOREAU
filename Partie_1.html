PARIE 1¶

Titre : Estimation de la tendance d’un problème de réflexion de Skorokhod associé à

une équation différentielle dirigée par le mouvement brownien fractionnaire et à un

processus de rafle de Moreau avec application en finance.
¶

Descriptif : Soit S(t) le prix d’un actif risqué à la date t, modélisé par l’équation différentielle stochastique :

\begin{align} dS(t)=\mu S(t) dt+ \sigma S(t) dB(t) \end{align}

Nous voulons simuler le mouvement brownien geometrique :¶

\begin{align} dS_{t}=\mu S_{t} dt+ \sigma S_{t} dB_{t} \end{align}
\begin{align} S0=s0 \end{align}

Si nous devons connaître la trajectoire de S. Nous allons utiliser la méthode d’Euler stochastique.
On va considérer une version discrétisée de cette équation, c'est à dire que nous avons l'approximer sous la forme:

\begin{align} \widehat{S}\equiv{s0} \end{align}
\begin{align} \widehat{S}_{(k+1)\Delta}\equiv \widehat{S}_{k\Delta t} + \mu \widehat{S}_{k\Delta t} + \Delta t \\ +\sigma \widehat{S}_{k\Delta t} \ (\widehat{B}(k+1)\Delta t \ - \widehat{B}_{k\Delta t}). \end{align}

seed = 5       
N  = 1000   #le nombre de périodes de temps

#### FONCTION

def Brownien(seed, N):
    
    np.random.seed(seed)   # pour toujours avoir les mêmes valeur  car sinon c'est aléatoire                    
    dt = 1./N                                    # Nombre de pas (tk = kT/N)
    b = np.random.normal(0, 1, int(N))#*np.sqrt(dt)  # #matrice composée de mxn variables aléatoires iid N(0,1)
    S = np.cumsum(b)                             
    return S, b



#### INTEGRER LES VALEUR
S = Brownien(seed, N)[0]
S = np.insert(S, 0, 0)                      # S0 = 0 pour le mouvement brownien
